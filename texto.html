Introdução font-properties

A tipografia transmite uma mensagem.
Introdução

Quando queremos dar uma ênfase no texto, nós podemos, por exemplo, escrever o mesmo em negrito.

Nós podemos transmitir uma mensagem diferente dependendo do estilo que escrevemos o texto. Ou, do tamanho.

CSS Fonts

Algumas das propriedades de fonts do CSS que podem nos ajudar a transmitir uma mensagem através dos textos da página são:

  • font-family;
  • font-weight;
  • font-style;
  • font-size;

Font-family

Tipo de fonte de um elemento.
O que ele faz?

Lista de fontes e ordem de prioridade e inclui um fallback

Fallback:

Fallback é: pular de um código para o outro, por exemplo:

font-family: 'Times New Roman', Times, serif;

No caso, o leitor procura a Times New Roman, quando não acha, procura por Times, quando não acha, aplica a serif.

Serifada:

Fonte com serifa.

Font-weight

Peso da fonte
Valores:

normal | bold | bolder | lighter | 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900

Dependendo da família da fonte não conseguimos utilizar todos os pesos de fonte.

Exemplo:
p {
font-weight:bold;
}

Web-fonts

Fontes armazenadas na web.
Fontes do sistema x Fontes da web

Fontes do sistema são as fontes que estão instaladas na máquina do usuário e nem sempre o cliente vai ter instalado as fontes usadas no projeto e isso pode fazer com que os estilos dos textos não sejam aplicados corretamente, mas para resolver esses casos podemos preparar nossas fonts para web ou usar fontes da web.

Como usar fontes para web?
  • @font-face;
  • @import;
  • link;
    • Como aplicar o CSS?

    Antes de aplicar o código para CSS, o link ou o @import da fonte deve estar no head do HTML.

    Font-variant e font-stretch

    Font Variant

    Faz variações na apresentação da fonte.

    p {
    font-variant: small-caps;
    }
    Font Stretch
    • Alargamento ou encolhimento da fonte;
    • Aceita palavras-chave como expanded, condensed, normal;
    • Aceita porcentagens de 50% a 200%;
    • Essa propriedade não vai funcionar em todas as fontes.
    p {
    font-stretch: expanded;
    }
    Referências:

    MDN | Font Variant

    MDN | Font Stretch

    MDN | Styling text Fundamentals

    HTML

    <p class="fontvariantp">
    Exemplo de Font Variant.
    </p>

    CSS

    p {
    font-variant: small-caps;
    }

    Exemplo de Font Variant.

    HTML

    <p class="fontstretchp">
    Exemplo de Font Stretch.
    </p>

    CSS

    p {
    font-stretch: condensed;
    }

    Line-height e text-transform

    Define os espaços entre linhas e transformação do texto.
    Line-height

    Pode ser com unidades ou sem unidades de medida.

    Valores comuns: 1.5 ou 2.

    p {
    line-height:1.5;
    }

    Referência:

    MDN | Line-height

    Text-transform

    Valores podem ser:

    none | capitalize | uppercase | lowercase | full-width | full-size-kana

    p {
    text-transform:uppercase;;
    }

    Referência:

    MDN | Text-transform

    Text-decoration

    Aparência decorativa de um texto.
    É possível aplicar mais de um valor.

    line: underline | overline | line-through

    style: wavy | dotted | double | dashed | solid

    color: values

    h1 {
    text-decoration: underline; /* Shorthand */
    }
    p {
    text-decoration: wavy overline blue; /* Shorthand */
    }

    Shorthand

    Colocar todos esses outros valores em uma única propriedade.
    Valores:

    font-style, font-variant, font-weight, font-stretch, font-size, line-height e font-family;

    p {
    font: italic normal bold normal 3em/1.5 Helvetica, Arial, sans-serif;
